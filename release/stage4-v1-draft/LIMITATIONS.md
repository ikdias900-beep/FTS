# Limitations

- This is a draft checkpoint capsule, not a public release, DOI archive, or final
  Stage 4 publication.
- The included grid is `fbt_atlas_v1_draft`; it is not a full finite atlas.
- The raw-cell artifact contains 144 cells from the draft config only.
- Aggregate values are `grid_frequency` summaries over the included raw cells only.
- The aggregate layer does not read the config or regenerate cells.
- Theorem 4 is not implemented, proved, reproduced, or reviewed as a theorem
  implementation.
- The capsule contains no stochastic simulation, evolutionary dynamics, ML/RL,
  dashboard, notebook, or figure output.
- The manifest copies retain original absolute paths from the packaging machine.
  The archived copies are validated through capsule-local checksums.
- The generated aggregate JSON retains the command template's conservative review
  caution. The authoritative checkpoint review status for this capsule is recorded in
  `REVIEW_REPORT.md` and `derived_data/stage4_atlas_v1_checkpoint.json`.
- No biological, real-perception, consciousness, spacetime, ontology, or metaphysical
  interpretation is licensed by this capsule.
