# Assumptions And Decisions

## Scientific Assumptions

### ASM-FFF-0001

Stage 1 public total-order outputs must show both:

- the source orientation-witness count;
- the distinct unique-function count.

The source formula counts preserving/reversing monotonicity witnesses. Finite unique-function counts differ because the admissible maximum constant function has both witnesses.

Status: `APPROVED` by Human PI on 2026-06-25 via `RDR-0002`.
