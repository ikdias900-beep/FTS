# Reproduction Commands

Run from the repository root.

## Environment Check

```bash
py -m uv run fts doctor --release-check
```

## Test Suite

```bash
py -m uv run ruff check .
py -m uv run ruff format --check .
py -m uv run mypy src
py -m uv run pytest
```

## Regenerate Stage 1 Artifacts

```bash
py -m uv run fts fff sweep --config experiments/configs/fff_stage1_small.json
py -m uv run fts validate-manifest experiments/manifests/<sweep-manifest>.json
py -m uv run fts fff publication-tables --sweep-manifest experiments/manifests/<sweep-manifest>.json
py -m uv run fts validate-manifest experiments/manifests/<publication-manifest>.json
```

The capsule artifacts were generated by:

```bash
py -m uv run fts fff sweep --config experiments/configs/fff_stage1_small.json
```
