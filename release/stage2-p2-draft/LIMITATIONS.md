# Limitations

## Scientific Scope

- Scope is limited to the FBT numerical appendix example in `SRC-FBT-2021`.
- The input domain contains only three world states, two observations, source priors,
  source likelihoods, and source fitness values.
- The implementation uses exact rational arithmetic only.
- The capsule does not include the general FBT theorem, finite FBT atlas, population
  dynamics, genetic algorithms, ML/RL, dashboards, or scientific figures.

## Interpretation Limits

- The exact values support only `CLM-FBT-APP-001` through `CLM-FBT-APP-004`.
- The result is an executable reproduction of one published table/appendix calculation,
  not a biological universality claim.
- No result in this capsule establishes a claim about real organisms, human perception,
  metaphysics, spacetime, consciousness, or conscious agents.
- General tie handling remains open under `ASM-FBT-0001`; this source example has unique
  MAP estimates.
- General zero-probability observation behavior remains open under `ASM-FBT-0002`; this
  source example has nonzero observation marginals.

## Release Limits

- This is a draft checkpoint assembled for repository inspection.
- It is not a GitHub release, DOI, archival deposit, or external publication by itself.
- It has not yet received a batched independent review.
- Project-authored files are MIT licensed; external primary sources and dependencies
  retain their own terms.
