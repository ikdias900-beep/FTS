# Reproduction Commands

Run from the repository root.

## Environment Check

```bash
py -m uv run fts doctor --release-check
```

## Test Suite

```bash
py -m uv run ruff check .
py -m uv run ruff format --check .
py -m uv run mypy src
py -m uv run pytest
py -m uv run fts validate-release-capsule release/stage3-p3-draft
```

## Regeneration Boundary

This capsule does not contain a generated scientific run artifact. To audit it:

1. compare `raw_data/stage3_structure_spec_cases.json` with
   `tests/fixtures/fff/stage3_structure_spec_cases.json`;
2. run the focused Stage 3 tests above;
3. validate the local capsule checksums.

The capsule checksum inventory is regenerated by hashing every committed file under
`release/stage3-p3-draft/` except `checksums.txt`.
