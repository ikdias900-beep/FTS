# Limitations

## Scientific Scope

- Scope is limited to the reviewed Stage 3 FFF permutation-group and measurable-space
  finite-helper bundle.
- The capsule packages specs, fixtures, helper implementations, and tests already
  reviewed under `REV-TASK-003-FFF-STRUCTURE-001`.
- The permutation-group work checks source-explicit finite helper behavior and formula
  arithmetic in the declared Stage 3 scope; it does not run exhaustive large
  symmetric-group searches.
- The measurable-space work uses finite partition bases and inverse-image
  measurability checks; it does not enumerate all finite algebra isomorphism classes.

## Interpretation Limits

- The measurable-space source expression is treated as an upper bound except in
  explicitly tested special cases where exact counts are independently enumerated.
- The permutation-group construction is not reduced to ordinary group homomorphisms.
- Counting-measure statements are not converted into claims about biological sampling
  distributions.
- No result here establishes a claim about real organisms, human perception,
  consciousness, spacetime, ontology, or conscious agents.

## Release Limits

- This is a draft checkpoint assembled for repository inspection.
- It is not a GitHub release, DOI, archival deposit, browser demo, figure package, or
  final public P3 publication.
- No new independent review is performed for this packaging task in the same session.
- Project-authored files are MIT licensed; external primary sources and dependencies
  retain their own terms.
